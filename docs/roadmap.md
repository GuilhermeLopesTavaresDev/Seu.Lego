# Roadmap

## Fase 1 - MVP

- Upload de imagem.
- Conversao para grade LEGO.
- Mapeamento para cores aproximadas.
- Previa visual.
- Lista de pecas `1x1`.
- Manual PDF simples.

## Fase 2 - Produto Usavel

- Cadastro e login.
- Historico de projetos.
- Reprocessamento com outros tamanhos.
- Melhor editor de recorte.
- Exportacao de lista de compras.
- Painel de fidelidade de cores.

## Fase 3 - Catalogo Real

- Sincronizacao com Rebrickable ou fonte equivalente.
- Cadastro local de cores LEGO.
- Cadastro local de pecas por cor.
- Filtros por disponibilidade.
- Estimativa de custo.

## Fase 4 - Otimizacao

- Agrupamento de pecas contiguas.
- Modo economico com pecas maiores.
- Comparacao entre fidelidade, custo e dificuldade.
- Manual por etapas mais detalhadas.

## Fase 5 - Monetizacao

- Previa gratis.
- PDF completo pago.
- Sistema de creditos.
- Planos mensais.
- Links afiliados para compra de pecas.
- Venda futura de kits prontos.
