# Monetizacao

## Modelo Inicial

O melhor modelo para inicio e freemium:

- Usuario envia foto e ve uma previa gratis.
- Download do PDF completo e lista detalhada de pecas fica pago.

## Possiveis Produtos Pagos

- Projeto avulso em PDF.
- Pacote de creditos.
- Plano mensal para criadores.
- Modo alta resolucao.
- Modo economico otimizado.
- Projetos comerciais para presentes personalizados.

## Evolucao

Quando houver volume, o sistema pode vender kits fisicos:

- O usuario paga pelo projeto e pelas pecas.
- O sistema separa ou terceiriza a separacao das pecas.
- O cliente recebe tudo pronto para montar.

## Cuidados Com Marca

O produto deve evitar parecer oficial da LEGO sem autorizacao. Termos como "compativel com pecas LEGO" ou "mosaico com pecas de montar" podem ser mais seguros dependendo da estrategia juridica.
